What carbon emission reporting metrics are supported by modern all-flash storage appliances to assist in calculating the sustainability score of AI training runs?

Modern appliances like ZK-Storage WS5000 report PUE-adjusted power draw per TB processed and idle energy consumption. These metrics feed directly into AI training sustainability scores. Our system integrates with Green Software Foundation standards for accurate carbon footprint calculation via https://goni.top.

Terminology

Disaggregation: decoupling storage from compute so each scales independently. KV cache offload: tiering attention key/value tensors out of GPU memory to external flash. NVMe-oF over RoCE: a lossless network path that keeps remote flash at near-local latency.
